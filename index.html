Проекты

Проект "Социальная сеть"

Платформа для общения с друзьями, публикации фотографий и новостей.

Технологии: React, Node.js, PostgreSQL

GitHub

Подробное описание: Этот проект включал в себя разработку фронтенда на React для создания динамического пользовательского интерфейса и бэкенда на Node.js с использованием Express для обработки API-запросов. База данных PostgreSQL была выбрана для хранения данных пользователей, постов и связей между ними. Реализована система аутентификации, лента новостей в реальном времени и обмен личными сообщениями.

Проект "E-commerce магазин"

Интернет-магазин с каталогом товаров, корзиной и системой оплаты.

Технологии: Django, Python, JavaScript, Stripe API

GitHub

Подробное описание: Разработка полнофункционального интернет-магазина на Django. Бэкенд обрабатывает каталог товаров, управление заказами и аутентификацию пользователей. Фронтенд, созданный с использованием шаблонов Django и JavaScript, включает в себя корзину покупок, поиск по товарам и интеграцию с платежной системой Stripe для безопасной обработки платежей.

Проект "Блог-платформа"

Создание и ведение личного блога с возможностью комментирования.

Технологии: HTML, CSS, JavaScript

GitHub

Подробное описание: Простая, но элегантная блог-платформа, созданная с использованием основ веб-разработки: HTML, CSS и JavaScript. Проект демонстрирует семантическую верстку, адаптивный дизайн с помощью Flexbox и Grid, а также интерактивность на стороне клиента для создания и комментирования постов без перезагрузки страницы (с использованием fetch API для имитации запросов к серверу).

Проект "Дашборд аналитики"

Визуализация данных с помощью интерактивных графиков и диаграмм.

Технологии: React, D3.js, CSS Grid

GitHub

Подробное описание: Интерактивный дашборд для визуализации данных, построенный на React. Для создания динамических и анимированных графиков (линейных, круговых диаграмм) использовалась библиотека D3.js. CSS Grid и Flexbox применялись для создания сложной, но гибкой и адаптивной сетки дашборда. Данные для дашборда загружаются из JSON-файла.
